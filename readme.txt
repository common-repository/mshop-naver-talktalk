== Description ==

대한민국에서 네이버와 카카오톡 아이디 없는 분 계신가요?

코드엠샵 소셜톡 플러그인은 워드프레스 쇼핑몰 코드엠샵에서 제공하는 대한민국 대표 웹메신저 플러그인으로 네이버 및 카카오톡 아이디로 워드프레스 및 우커머스 쇼핑몰에서 판매자와 고객이 소통할 수 있는 웹채팅 메신저 서비스 입니다.

<a href="https://bit.ly/3h8gV1K" target="_blank">플러그인 설치 영상 매뉴얼 : https://bit.ly/3h8gV1K</a>
<a href="https://bit.ly/3jK0Wsw" target="_blank">플러그인 텍스트 매뉴얼 : https://bit.ly/3jK0Wsw</a>

[카카오 상담톡 상세 기능]
- 카카오톡을 이용한 실시간 채팅 (PC, 모바일 상담)
- 플러스 친구 등록 : 메시지 발송을 통한 마케팅 확장 가능
- 통계 : 성별, 연령, 유입경로, 문의 통계 기능
- 단체 문자 발송, 쿠폰, 메시지 발송 및 예약 발송, 그룹 관리
- 상담 운영 설정 : 채팅 운영 요일, 시간 설정, 자주쓰는 답변 기능

[네이버톡톡 상세 기능]
- 네이버 아이디를 이용한 실시간 채팅 (PC, 모바일 상담)
- 채팅 메시지 분석을 통해 대상 지정 및 단체 문자 발송
- 쿠폰, 명함, 설문지 발송 및 예약 발송
- 통계 : 성별, 연령, 유입경로, 문의 통계 기능
- 상담 운영 설정 : 채팅 운영 요일, 시간 설정, 자주쓰는 답변 기능
- 챗봇설정 관리

== Installation ==

1. 워드프레스 플러그인 화면에서 엠샵 소셜톡을 검색하여 직접 설치하거나, 다운로드 받은 파일을 /wp-content/plugins/mshop-naver-talktalk 디렉토리에 업로드하여 설치 합니다.
2. 엠샵 소셜톡 플러그인을 활성화 합니다.
3. 엠샵 소셜톡 메뉴에서 플러그인 설정을 진행 합니다.

== Frequently Asked Questions ==

= 플러그인 설치만으로 ‘네이버톡톡’ 및 ‘카카오플러스 친구’ 를 바로 이용할 수 있나요? =

카카오 플러스친구(https://business.kakao.com/info/plusfriend) 가입 후 플러그인 설치 연동을 통해 바로 이용하실 수 있으며,
네이버톡톡은 파트너센터(https://partner.talk.naver.com) 가입 후 검수 승인 절차를 거쳐 이용이 가능합니다.
